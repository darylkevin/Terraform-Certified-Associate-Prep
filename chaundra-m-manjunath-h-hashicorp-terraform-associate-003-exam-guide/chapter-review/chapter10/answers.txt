**Question 3**

Option D is correct because there is no provisioner called a cloud provisioner supported by Terraform currently. Option A is incorrect because Terraform supports the file provisioner to create new files during the Terraform execution on the local machine. The local-exec and remote-exec provisioners are used to execute commands on the local machine running Terraform and the resource created remotely by Terraform, respectively. Hence, options B and C are also incorrect.
